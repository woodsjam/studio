# **App Name**: CertifyAI

## Core Features:

- Google Sign-In: Secure login with Google SSO.
- Continuous Audio Recording: Record audio continuously during the interview session using MediaRecorder.
- Live Transcript UI: Display the transcribed text in real-time, preventing copy and pasting via CSS and zero-width characters.
- Enforce Linear Progression: Implement linear progression through the interview questions, disallowing going back to previous questions.
- QR Identity Link: Generate dynamic links for faculty/proctors to auto-link their identity to the interview session using QR codes.
- Summarization and Anomaly Detection: Employ AI tool: Summarization & timing anomaly hints (diagnostic only) (Firebase AI Logic (Gemini) for summarization & timing anomaly hints (diagnostic only).
- Wrap-Up Amendments: Enable amendments during the Wrap-Up phase with append-only changes to the transcript.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and professionalism, aligning with the certification theme.
- Background color: Light gray (#F5F5F5), offering a neutral backdrop that emphasizes content clarity and reduces visual strain.
- Accent color: Soft green (#8BC34A), signaling accuracy and endorsement, used for marking completion or certification milestones.
- Body and headline font: 'Inter' sans-serif font for a modern, machined, objective and neutral look.
- Use minimalist icons to represent different actions (e.g., recording, linking identity) for ease of use.
- Implement a clean and structured layout with clear visual hierarchy to guide users through the interview process.
- Add subtle animations for transitions and loading states to improve user experience.
